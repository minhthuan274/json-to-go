SON-BSON-to-Go

Convert JSON/BSON to Go struct

This tool instantly converts JSON|BSON into a Go type definition. Paste a JSON structure on the left and the equivalent Go type will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output!
